# Gerenciador de Carreira — Vale Produção

Site/app responsivo (mobile-first) para diagnóstico de carreira musical com relatório premium.

## Rodar localmente
Basta abrir `index.html` no navegador.

> Dica: se quiser um servidor local (recomendado para modules), use:
- VSCode Live Server, ou
- `python -m http.server 8080` dentro da pasta do projeto

## Publicar no GitHub Pages
Settings → Pages → Deploy from a branch → `main` → `/root`.
